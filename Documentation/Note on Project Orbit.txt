Installation & Setup
1. Install Dependencies

Make sure the required libraries are installed. On Linux (Ubuntu/Debian), run:

sudo apt update
sudo apt install build-essential cmake libgl1-mesa-dev libx11-dev libxrandr-dev \
libxinerama-dev libxcursor-dev libxi-dev libxkbcommon-dev

Explanation:

build-essential → compiler tools (gcc, g++, make)

cmake → to generate build files

libgl1-mesa-dev → OpenGL development files

libx11-dev, libxrandr-dev, libxcursor-dev, libxi-dev → X11 libraries needed by GLFW

libxkbcommon-dev → for keyboard handling

2. Clone or Download Project
git clone <your-repo-url>
cd ProjectOrbit
3. Build the Project
mkdir Build
cd Build
cmake ..
cmake --build .

This will generate your executable in the Bin/ folder.

4. Run the Program
../Bin/ProjectOrbit

Or if you’re in the project root:

./Bin/ProjectOrbit
Optional: Clean Build

If you ever want to rebuild from scratch:

rm -rf Build
mkdir Build && cd Build
cmake ..
cmake --build .

💡 Tip: You can also add a dependencies script (install_deps.sh) to make it one command for Linux users.
